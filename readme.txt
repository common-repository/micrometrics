== Description ==
Simple and friendly web analytics. Track 100% of the visits, respecting the privacy of your users, and continue making the best decisions for your business.

MicroMetrics is an alternative (or add-on) to Google Analytics that does not compromise visitor privacy by not tracking IPs, fingerprints, or cookies. A twist on website analytics that offers ease of use and respect for privacy.

This plugin adds the MicroMetrics tracking code to your website header quickly and easily, just by activating the plugin. You need a [MicroMetrics](https://micrometrics.es "Web analytics without complexes") account before installing the plugin in order to collect website statistics.

If you have any questions, [contact us](https://micrometrics.es/contacto/ "Contact us").

== Installation ==
1. You can:
  * Upload the folder to the */wp-content/plugins/* directory via FTP.
  * Upload the complete ZIP file via *Plugins -> Add New -> Upload* in the Admin Panel of your WordPress installation.
  * Search for **micrometrics** in the search engine available in *Plugins -> Add New* and press the *Install Now* button.
2. Activate the plugin through the *Plugins* menu in the WordPress Administration Panel.
3. That's it!
